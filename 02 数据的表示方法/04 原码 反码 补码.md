# 原码 反码 补码
### 为什么会有原码 反码 补码
```
    早期8086CPU(16位处理器)只支持加法指令，
    减法、乘法、除法都是通过加法指令实现的。
    
    简化计算机集成电路的设计
```
## 原码 反码 补码定义
### 原码
```
    原码：最高位为符号位，（“ 0 ”代表正数，“ 1 ”代表负数）
    其余各位为数值本身的绝对值。
``` 
### 反码
```
    正数：反码与原码相同，这是规定。
    负数：符号位为 1 不变，其余位对原码取反。
```
### 补码
```
    正数：补码与原码相同。
    负数：符号位为 1 不变，其余位对原码取反加 1 。
```
