# 02 位运算之加减乘除
### 左移运算 shift left(shl <<)
```
    全部向左移动，高位舍弃，低位补0
    左移一位，相当于乘以2，左移两位乘以4
```

### 右移运算 shift right(shr >>)
```
    全部向右移动，低位舍弃，高位补0
    右移一位，相当于除以2，右移两位除以4
```
